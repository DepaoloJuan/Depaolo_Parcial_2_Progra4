# Depaolo_Parcial_2_Progra4

Preguntas para el docente:

1. Perfil administrador en el registro — El enunciado del sprint 1 dice que el perfil es "usuario" por defecto pero se puede modificar para ser "administrador". ¿Esto significa que cualquier usuario puede auto-asignarse el perfil administrador desde el formulario de registro? ¿O el perfil administrador solo lo puede asignar otro administrador desde el dashboard del sprint 4?

2. Comentarios en Mi Perfil (sprint 2) — El sprint 2 pide mostrar las últimas 3 publicaciones del usuario "y sus comentarios", pero el módulo de comentarios se implementa en el sprint 3. ¿Alcanza con mostrar las publicaciones sin comentarios en el sprint 2, y agregar los comentarios cuando esté el módulo listo en el sprint 3?

3. Editar perfil — El enunciado general dice "Cada usuario debe poder ver y editar su perfil", pero en ningún sprint específico se pide la funcionalidad de editar. ¿En qué sprint corresponde implementarlo?

4. Página de detalle de publicación — El sprint 3 pide una pantalla de detalle donde se ven los comentarios con paginación. ¿Es una página separada (/publicaciones/:id) o es un componente que se expande dentro de la página de publicaciones?
